# Curses Tic Tac Toe
**Tic Tac Toe game built using ncurses in C**

## Building
- Requirements
  - Clang v9.0.0 or newer
  - Ncurses

- Clone this repo
```bash
git clone https://github.com/PrajwalCH/curses_tic_tac_toe.git
cd curses_tic_tac_toe
```
- Run make
```bash
# For debug build run with DEBUG=1
make
```
